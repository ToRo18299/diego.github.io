## Task Tracker

Task Tracker is a personal task management application built with Vite, React, and TypeScript. It is a 100% frontend solution that leverages localStorage for data persistence and uses mock data for the initial state. The app features a modern, professional UI based on Material UI (MUI), supports both dark and light modes, and includes subtle animations for a smooth user experience.

### Features

- Fully frontend, no backend or server dependencies
- CRUD operations for tasks and projects using localStorage
- Dark/Light mode with theme switch
- Subtle animations for adding, editing, and deleting tasks
- Filters by project, status, and order
- Modern, professional UI with MUI
- Modular, clean, and well-documented codebase

### Local Installation

1. Clone the repository
2. Install dependencies: `npm install`
3. Run in development mode: `npm run dev`
4. Open [http://localhost:5173](http://localhost:5173)
